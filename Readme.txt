Descripción:

Este proyecto utiliza inteligencia artificial para ayudar a conductores sordos a identificar vehículos de emergencia, como ambulancias y policía, mediante el reconocimiento de sonidos. La tecnología determina la posición relativa de estos vehículos, permitiendo a los conductores tomar decisiones seguras en la carretera. La plataforma incluye una interfaz amigable que brinda acceso rápido a esta información crucial. El objetivo es mejorar la seguridad y la independencia de las personas sordas en situaciones de tráfico y su vida cotidiana.

Instrucciones:

Antes de lanzar el codigo de FrontEnd, hay que ejecutar el codigo del back escribiendo en la consola 'node server.js'. Una vez el back está en funcionamiento, habría que ejecutar el front con 'python manage.py runserver', si se requiere, instalar los modulos solicitados para el correcto funcionamiento del codigo.
